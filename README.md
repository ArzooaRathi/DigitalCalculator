# Digital Calculator & Login System

This project is developed using Java and Java Swing in NetBeans IDE.

## Features
- Digital Calculator
- Login Page
- Signup Page
- GUI built using Java Swing

## Technologies Used
Java
Java Swing
MySQL
JDBC Connectivity
NetBeans IDE


## How to Run
1. Open the project in NetBeans.
2. Run the main class
3. .Before running the project, import the mydb.sql file into MySQL Workbench.
